# 关于印发进一步规范砂石料矿开采安全监管工作的通知

 

各地、州、市应急管理局，相关企业：

为深入贯彻《中共中央办公厅 国务院办公厅关于进一步加强矿山安全生产工作的意见》和《国家矿山安全监察局关于印发〈关于加强非煤矿山安全生产工作的指导意见〉的通知》（矿安〔2022〕4号）等有关规定，进一步规范砂石料矿安全生产行政许可和安全监管事项，压紧压实安全监管部门安全监管责任和企业安全生产主体责任，推动全区砂石料矿开采企业安全有序规范开采，有效遏制各类事故发生，现就有关事项通知如下：

一、规范安全设施设计审查和安全生产许可证审核颁发

（一）强化建设项目安全设施设计管理

砂石料矿新建、改建、扩建建设项目应严格执行安全设施“三同时”制度，委托有资质的设计单位编制安全设施设计。采矿许可证范围内的矿产资源原则上应当进行一次性总体安全设施设计，由一个生产经营单位统一管理，原则上只能设置一个独立生产系统，独立生产系统设计生产规模和服务年限应当达到国家、自治区规定的最低标准。

（二）规范建设项目安全设施设计审查

新建、改（扩）建项目设计年产1000万吨及以上或者采场边坡高度200米及以上的砂石料矿建设项目，要依据《国家安全监管总局办公厅关于明确非煤矿山建设项目安全监管职责等事项的通知》（安监总厅管一〔2013〕143号），报请国家矿山安全监察局审批。其他砂石料矿建设项目安全设施设计的审查由砂石料矿开采企业自行组织，形成审查报告备查，参与审查的专家不少于5名，应当包含采矿、地质、机电等专业，且具有副高及以上工程师资格。

（三）全面实行分级分类安全生产许可证审核颁发

砂石料矿实施分级分类安全生产许可，自治区应急管理厅负责采场设计边坡高度150米及以上或年生产规模1000万吨及以上以及中央管理企业所属的砂石料矿的安全生产许可证审核颁发。各地州市应急管理局负责采场设计边坡高度小于150米且年生产规模小于1000万吨的砂石料矿开采企业安全生产许可证审核颁发。采场设计台阶高度不超过5米且设计边坡高度不超过20米的砂石料矿不实施安全生产许可。本通知印发前存续的砂石料矿开采企业应于2025年3月31日前依法取得安全生产许可证。

二、严格安全和技术管理

砂石料矿开采企业应建立健全覆盖实际控制人在内的全员安全生产责任制和岗位安全操作规程。依法设立安全管理机构或者配备专职安全生产管理人员，专职安全生产管理人员应具有相应的非煤矿山安全生产专业知识和工作经验，数量不低于从业人员的1%，不少于2人。配备具有矿山主体专业中专及以上学历或者中级及以上技术职称的技术人员，配齐特种作业人员，并持证上岗。严格按照设计组织生产建设，落实评价、设计、建设、施工、监理各方安全责任，严禁以采代建，严禁掏采或者“一面墙”开采。要常态化开展隐蔽致灾因素普查治理，现状高度150米及以上的边坡和排土场应当进行在线监测，现状高度100米及以上的边坡和排土场，应当每年进行一次边坡稳定性分析。依法加强安全生产标准化管理体系建设，建立健全安全风险分级管控和事故隐患排查治理双重预防机制，严格开展风险辨识评估并实施分级管控，定期开展全员全覆盖隐患排查治理，持续提高企业安全管理水平，确保安全生产。

三、落实安全分级属地监管

各地应急管理部门要严格落实日常安全监管职责，中央管理企业所属砂石料矿、采场设计边坡高度超过150米或年开采能力达到1000万吨及以上的砂石料矿开采企业及其上级公司（自治区负责日常安全监管的除外）由地州市应急管理部门负责日常安全监管，其他砂石料矿开采企业由县市区应急管理部门负责日常安全监管。各地应急管理部门要按照《国家矿山安全监察局关于印发非煤矿山安全风险分级监管办法的通知》（矿安〔2023〕1号）规定，每年科学评定风险等级，确定执法检查频次，制定年度检查计划，严格落实日常监管执法职责。依法打击未依法取得采矿许可证、安全生产许可证擅自开采和以采代建等违法违规行为，对存在未履行安全设施“三同时”、不按设计施工开采、安全技术管理人员缺失等问题隐患的，坚决依法予以停产停业整顿。对存在重大事故隐患，一百八十日内三次或者一年内四次受行政处罚的，经停产停业整顿仍不具备安全生产条件的或拒不执行负有矿山安全生产监督管理职责部门作出的停产停业整顿决定的，依法提请县级以上人民政府予以关闭。

本通知自印发之日起执行，原自治区安全生产监督管理局印发的《自治区安全监管局关于2018年取消部分砂石料场安全生产许可事项的通知》（新安监非煤〔2018〕83号）同时废止。
