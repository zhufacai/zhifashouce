# 新疆维吾尔自治区烟花爆竹经营（批发）企业布点规划

 

为从源头上加强烟花爆竹经营（批发）环节的安全监管，推动烟花爆竹行业规范和安全发展，按照《烟花爆竹安全管理条例》（国务院第455号令）、《烟花爆竹经营许可实施办法》（国家安监总局第65号令）、《关于进一步加强烟花爆竹安全监督管理工作的意见》和《“十四五”国家安全生产规划》等有关法律法规规定和规划要求，规范自治区烟花爆竹经营（批发）企业布点有关事项，制定本规划。

一、总体目标

按照“保障安全、统一规划、合理布局、总量控制、适度竞争”的原则，综合考虑各地、州、市区域面积、人口总量、市场需求、环境容量、风俗民情、燃放要求以及现有烟花爆竹经营（批发）企业数量等情况，科学合理布设规划，构建烟花爆竹经营（批发）网络，健全全区烟花爆竹安全监管体系，全面强化烟花爆竹安全、规范、稳定、有序经营的长效机制。

二、布点要求

通过对自治区烟花爆竹经营（批发）企业现状及需求分析，确定下列规划布点要求：

（一）喀什地区、和田地区和克孜勒苏柯尔克孜自治州以城镇人口数量100万（含本数）为基数，其他地区以城镇人口数量超过45万（含本数）为基数，城镇人口数量超过基数的地、州、市可规划设置1个烟花爆竹经营（批发）企业布点数。以此类推，各地、州、市新增烟花爆竹经营（批发）企业布点数与城镇人口数量成比例增加。

（二）各县（市、区）政府所在地相互之间超过200公里（含本数）供货距离，即可新增1个批发企业布点数。

（三）各地、州、市当城镇人口数量和供货距离达到设置1个烟花爆竹经营（批发）企业布点数要求时，不再重复设置。当各地、州、市城镇人口数量和供货距离与设置布点数发生较大变化时，可根据人口数量和供货距离设置要求布点数总和的平均值设置。

（四）依据《反垄断法》和《公平竞争审查条例》的相关规定，各地、州、市设置不少于2个烟花爆竹经营（批发）企业布点数。

（五）各地、州、市烟花爆竹经营（批发）企业数量原则上不超过《自治区烟花爆竹经营（批发）企业布点分配表》所规定的布点数，各地、州、市如需调整布点数，可根据实际需要向自治区应急管理厅提出申请，经批准后可调整烟花爆竹经营（批发）企业布点数。

（六）执行动态管理退出机制。各地、州、市对现有企业有条件的可以整合；自行申请退出（不含申请变更）的，２年内未申请取得《烟花爆竹经营（批发）许可证》的，可予以替补。

三、布点分配

按照地域和人口分布以及现有数量，合理规划设立烟花爆竹经营（批发）企业布点数，在现有基础上增加55.2%烟花爆竹经营（批发）企业布点数，总数控制在45个左右。
