# 国务院安全生产委员会关于印发《涉及危险化学品安全风险的行业品种目录》的通知

时效性：    现行有效

发文机关：  国务院安全生产委员会

文号：      安委〔2016〕7号

发文日期：  2016年06月28日

施行日期：  2016年06月28日

效力级别：  部门规范性文件

各省、自治区、直辖市及新疆生产建设兵团安全生产委员会，国务院安全生产委员会各成员单位，各有关中央企业：

为深刻吸取天津港“8·12”瑞海公司危险品仓库特别重大火灾爆炸等事故的教训，落实有关事故防范措施，有效防范和遏制危险化学品重特大事故，国务院安委会组织研究编制了《涉及危险化学品安全风险的行业品种目录》，用于指导各地区和各有关行业全面摸排涉及危险化学品的安全风险。经国务院领导同志同意，现印发给你们，请结合《国务院安全生产委员会关于深刻吸取天津港“8·12”特别重大事故教训集中开展危险化学品安全专项整治的通知》（安委〔2016〕4号），联系各地区、各行业实际，深入排查涉及危险化学品的安全风险，建立危险化学品安全风险分布档案，有效防控危险化学品安全风险。
